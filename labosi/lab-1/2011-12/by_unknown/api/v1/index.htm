Filminfo

Servis za upravljanje i pronalaženje informacija o filmovima.

REST API sredstva - V1

Slijedi opis svih sredstava unutar pojedinih grupa sredstava. Sva sredstva primaju i vraćaju informaciju u Json formatu.

Directors

CRUD Sredstvo URL sredstva Opis
GET /directors http://host:port/Filminfo/directors Dohvaća listu svih postojećih redatelja.
GET /directors/ime http://host:port/Filminfo/directors/ime Dohvaća sve redatelje sa danim prvim imenom. Ako takvih nema, vraća se prazna lista.
GET /directors/_prezime http://host:port/Filminfo/directors/_prezime Dohvaća sve redatelje sa danim prezimenom. Ako takvih nema, vraća se prazna lista.
GET /directors/ime_prezime http://host:port/Filminfo/ime_prezime Dohvaća redatelja sa danim imenom i prezimenom. Ako takvog nema, vraća se prazni skup.
POST /directors http://host:port/Filminfo/directors Primljeni podatak (redatelj) postavlja se u bazu redatelja.
PUT /directors/ime_prezime http://host:port/Filminfo/ime_prezime Redatelj sa danim imenom i prezimenom zamjenjuje se primljenim podatkom (redatelj).
PUT /directors/id http://host:port/Filminfo/directors/id Redatelj sa oznakom id zamijeniti će se primljenim podatkom (redatelj).
PUT /directors http://host:port/Filminfo/directors Skup redatelja zamijeniti će se primljenim podatcima (redateljima) na temelju njihovih oznaka.
DELETE /directors/ime_prezime http://host:port/Filminfo/ime_prezime Uklanja redatelja sa danim imenom i prezimenom i sve filmove koje je on režirao.
DELETE /directors/id http://host:port/Filminfo/directors/id Uklanja redatelja sa unesenom oznakom i sve filmove koje je on režirao.

Films

CRUD Sredstvo URL sredstva Opis
GET /films http://host:port/Filminfo/films Dohvaća listu svih postojećih filmova.
GET /films/id http://host:port/Filminfo/films/id Dohvaća film prema zadanoj oznaci id. Ako takav ne postoji vraća se prazan element.
GET /films/d_id http://host:port/Filminfo/films/d_id Dohvaća listu svih filmova ćiji redatelj ima oznaku id. Ako ne postoji niti jedan takav film, vraća se prazna lista.
GET /films/ime_prezime http://host:port/Filminfo/films/d_id Dohvaća listu svih filmova ćiji je redatelj danog imena i prezimena. Ako ne postoji niti jedan takav film, vraća se prazna lista.
POST /films/ime_prezime http://host:port/Filminfo/films Primljeni podatak (film) postavlja se u bazu filmova, a kao redatelj uzima se onaj sa danim imenom i prezimenom.
PUT /films/id http://host:port/Filminfo/films/id Film sa oznakom id zamjenjuje se primljenim podatkom (filmom).
PUT /films http://host:port/Filminfo/films Skup filmova zamijeniti će se primljenim podatcima (filmovima) na temelju njihovih oznaka.
DELETE /films/id http://host:port/Filminfo/films/id Uklanja film sa unesenom oznakom.
